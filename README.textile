Comix [It's a Trap!] is an addon for World of Warcraft (WoW).

Adds sound for "traps":http://www.wowhead.com/?search=trap#abilities to "Comix":http://wow.curse.com/downloads/wow-addons/details/comix-the-return.aspx.

Usage: use any trap ability

Note that "Comix":http://wow.curse.com/downloads/wow-addons/details/comix-the-return.aspx is an optional dependency for this addon. It is designed to use "Comix":http://wow.curse.com/downloads/wow-addons/details/comix-the-return.aspx settings if present, but will function without it installed.
